# Spoofer

Утилита на языке C# (WinForms), предназначенная для изменения идентификационных данных операционной системы Windows через редактирование системного реестра.

## Функциональные возможности
* Изменение MachineGuid (HWID) в разделе Cryptography.
* Генерация и замена Product ID (в формате OEM).
* Изменение сетевого имени компьютера (Computer Name).
* Редактирование имени зарегистрированного владельца системы (Registered Owner).
* Автоматическая генерация случайных значений для всех полей.
* Отключение проверки TPM по нажатию кнопки.

## Системные требования и ограничения
* Наличие прав администратора: работа с веткой HKEY_LOCAL_MACHINE требует повышенных привилегий.
* Перезагрузка: изменения вступают в силу только после полной перезагрузки ОС.
* Платформа: .NET 6.0 или выше.

## Отказ от ответственности
Данное программное обеспечение предоставляется «как есть». Автор не несет ответственности за стабильность работы системы или стороннего лицензионного ПО после изменения системных идентификаторов. Используйте утилиту на свой страх и риск.

## Сборка и установка
1. Клонируйте репозиторий.
2. Откройте проект в Visual Studio 2022.
3. Установите конфигурацию сборки на Release.
4. Выполните сборку решения (Build Solution).

## Порядок использования
1. Запустите исполняемый файл от имени администратора.
2. Используйте кнопку «Сгенерировать» для заполнения полей случайными данными или введите значения вручную.
3. Нажмите кнопку «Сохранить» для записи данных в реестр.
4. Перезагрузите компьютер.
